Програм хөгжүүлэгч - Голомт Банк (2024 оны 5-р сараас - Одоо)

  • SocialPay багийн backend хөгжүүлэлтэд оролцож, урьдчилан сэргийлэх засвар үйлчилгээ болон алдааг шийдвэрлэх замаар аппликешний тогтвортой, найдвартай байдлыг хангасан бөгөөд банкны системийн аюулгүй байдлын протоколын талаар практик туршлага хуримтлуулсан.
  • Spring Boot ашиглан ЖДҮ-д зориулсан банкны аппликешний цогц backend системийг бие даан хөгжүүлж, нэвтрэх эрх баталгаажуулалт, гүйлгээ боловсруулалт болон ПОС терминалын харилцаа холбоог интернэт банктай аюулгүй интеграци хийж хэрэгжүүлэн, сууриас нь эхлэн өргөтгөх боломжтой системийн архитектур зохиомжлох мэдлэг эзэмшсэн.
  • Oracle өгөгдлийн сан ашиглан SocialPay аппликешний мэдэгдлийн түүхийн функцыг зохион бүтээж, өндөр хэмжээний өгөгдөл удирдахад зориулж query гүйцэтгэл болон хадгалалтын үр ашгийг оновчтой болгож, өндөр ачаалал бүхий аппликешнд зориулсан өгөгдлийн сангийн үр дүнтэй удирдлагын ойлголтоо гүнзгийрүүлсэн.
  • SocialPay болон MostMoney платформуудын хооронд QR кодын харилцан ажиллах боломжийг амжилттай интеграци хийж, баг хоорондын хамтын ажиллагаа болон гуравдагч талын интеграцийн өндөр ур чадварыг харуулж, платформ дундын интеграци болон API хөгжүүлэлтийн туршлага хуримтлуулсан.
